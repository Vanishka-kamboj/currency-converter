# currency-converter
Real-time currency converter using JavaScript Fetch API and exchange rate API integration.

# Features
1.Convert between multiple currencies
2.Real-time exchange rate fetching using API
3.Country flags auto-update based on selected currency
4.Clean and responsive UI
5.Instant conversion result display

# Tech Stack
HTML5 – Structure
CSS3 – Styling & Layout (Flexbox)
JavaScript (ES6) – Logic & API Integration
Exchange Rate API (for live data)


# How to Run Locally
1.Clone this repository:
git clone https://github.com/your-username/currency-converter.git
2.Open the project folder.
3.Open index.html in your browser.
